Network Arch & Protocols


Instructor: Dr Zeina Awada


CRN: 11689


Meeting Time : TTR 8:00 AM - 09:15 AM


Class : Block H|Room 106


Syllabus: This course introduces the fundamental concepts of computer networks and network architectures, focusing on network models, protocols, and real-world communication systems. Students will understand how data is transmitted, routed, and secured over various network layers.

Finite Element Method


Instructor: Dr Elias Farah


CRN: 10759


Meeting Time : TTR 11:00 AM - 12:15 PM


Class : Block H|Room 004


Syllabus: This course provides an introduction to the theory and application of the Finite Element Method for solving differential equations arising in engineering and physical sciences. It emphasizes the mathematical foundation, formulation of problems, and implementation techniques.

Math For Engineers


Instructor: Dr Rita Harb


CRN: 10613


Meeting Time : MF 2:00 PM - 3:15 PM


Class : Block H|Room 107


Syllabus: This course covers essential mathematical tools and techniques used in engineering, including calculus, linear algebra, differential equations, complex numbers, and basic probability. Emphasis is placed on solving engineering problems using mathematical reasoning and analytical methods.

×

Web Programming


Instructor: Ing Pascal Damien


CRN:11212


Meeting Time : MF 3:00 PM - 2:45 PM


Class : Block H|Room 107


Syllabus: This course introduces the fundamental technologies used in developing modern, dynamic websites and web applications. Students will learn both frontend and backend development, covering HTML, CSS, JavaScript, server-side scripting, databases, and modern frameworks.

Non Linear Electronics


Instructor: Dr Joseph Al Assad


CRN: 10615


Meeting Time : TTR 12:30 PM - 1:45 PM


Class : Block H|Room 107


Syllabus: This course introduces students to non-linear electronic devices and circuits, focusing on their analysis, design, and applications. Topics include diode and transistor circuits, waveform shaping, oscillators, amplifiers, and non-linear system modeling.

Non Linear Electronics Lab


Instructor: Mr Roy Samia


CRN: 10944


Meeting Time : MF 5:30 PM - 7:20 PM


Class : Block Hs|Room 126


Syllabus: To provide hands-on experience with non-linear electronic devices and circuits, enabling students to analyze, design, and test non-linear behaviors using components like diodes, transistors, and op-amps.

Network Arch & Protocols Lab


Instructor: Dr Diala Abi Haidar


CRN: 12981


Meeting Time : W 9:00 AM - 10:40 AM


Class : Block Hs|Room 103


Syllabus: Students will learn to design and implement small-scale networks, configure routers and switches, use network simulation tools (such as Cisco Packet Tracer or Wireshark), and analyze protocol behavior across different layers of the OSI model.
